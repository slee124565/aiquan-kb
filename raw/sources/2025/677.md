# 677｜周日荐书：从《至高无上》看一个点赞背后，两个AI巨头的多年战争

2025年11月23日

你好，我是快刀青衣。欢迎收听快刀广播站，每天带你看AI。

又到了周日荐书环节。最近半个月，AI圈几乎被谷歌霸屏了，先是性能强悍的Gemini 3 Pro，我在昨天的第676期广播站就专门评测了这个模型；其次是之前就引发极大关注的谷歌图像模型Nano Banana，这次也推出了Pro版本。

我在两年前就有一个想法：让AI直接把80后经典游戏《街头霸王》里的角色，变成真人版模特海报。但是这两年，每有一个新模型的发布，我都会尝试这个生图想法，却一直没能实现。没想到，这**次Nano Banana Pro完美完成了任务，甚至连每个人物背后的实景，都是AI自己推理设计的**。因为我的提示词很随意，关于背景只写了一句“你自己想想角色应该站在什么地方”。感兴趣的同学可以点击文稿区末尾的链接查看这组图片，也可以关注我的公众号“快刀青衣”。

说实话，**这次Gemini 3 Pro的发布特别出人意料**。在当地时间11月18日（北京时间11月19日凌晨），谷歌没有漫长的预热，没有概念片，直接上线模型、开放API、推出应用平台——这种发布方式在科技圈挺少见的。大公司通常喜欢造势，提前几周预告，开发布会，请媒体，搞得热热闹闹。**但这次谷歌像突然按下红色核按钮，直接扔出了一个王炸。**

更有意思的是，OpenAI掌门人山姆·奥尔特曼，给这次发布点了个赞。

你想想，这是什么场景？你的竞争对手发布产品，而且性能数据相当炸裂——数学测试满分，屏幕理解能力是你的20倍，商业模拟能力是你的3.7倍。按理说，你要么装作没看见，要么让公关团队赶紧研究怎么应对。但奥尔特曼没有，他直接点了个赞。

这个点赞让我挺好奇的。也正是因为这个点赞，我很想去了解下两家公司背后的故事，于是我就找到了帕米·奥尔森写的这本书，名叫**《至高无上：一场颠覆世界的人工智能竞赛》。**

这本书用了大量篇幅讲述OpenAI和DeepMind的故事，也就是山姆·奥尔特曼和德米斯·哈萨比斯这两个男人的故事。单听书的标题，有点像一本有点无聊的人工智能综述，但我读完才明白，**这个点赞背后，其实是一场持续十年的战争**。而这场战争的起点，要从两人截然不同的童年说起。

哈萨比斯11岁那年，参加了一场国际象棋锦标赛。他从小痴迷象棋，但那场比赛他输了。这次失败让他突然顿悟：**人类的战略思维不应该只用在棋盘上，应该用在更重要的地方。**

从那以后，哈萨比斯迷上了游戏开发。他想做的不是普通游戏，而是能让玩家“扮演上帝”的模拟经营类游戏。在游戏里，你可以设计无数贴近现实的细节，掌控整个世界的运转。**这种对“模拟现实”的痴迷，后来也变成他对AI的执念**，他想用AI模拟宇宙，揭示宇宙的运行机制。

奥尔特曼的童年完全不同。他是在美国中部长大的同性恋少年，从“美国在线”的聊天室里找到了群体归属感。他从小就有很强的使命感，在学校里尽可能多地担任领导角色，为性少数群体争取权利。

但真正改变奥尔特曼的，是他到硅谷后对死亡的恐惧。他开始思考，如何才能永久留存人类意识？他甚至还在家里准备了枪支和防毒面罩，同时开始相信，**自己会成为引导人类走向“人工智能乌托邦”的旗手。**

你看，**两人的起点不同，但目标却惊人地一致：都想造出改变人类命运的AI。**

书里用了大量笔墨讲了哈萨比斯的理想主义，从DeepMind早期的运营细节就能看出来，有些细节读起来甚至有点魔幻，我来和你讲几个例子。

在DeepMind早期找投资人的时候，哈萨比斯把自己的理念提炼成DeepMind的宣传口号：“破解智能，并用智能解决其他一切问题”，还做成幻灯片展示给投资者。但另一位联合创始人穆斯塔法·苏莱曼不赞同这个说法。有一天趁哈萨比斯不在，苏莱曼让一名资深员工修改幻灯片，最后口号变成了“破解智能，并用智能让世界变得更美好”。

哈萨比斯很不喜欢修改后的口号。后来他回到办公室，又让同一名员工改了回去，“用智能解决其他一切问题”。老实说，两个联合创始人意见不合，却通过让员工改PPT的方式做较量，两人真是用最“英国化”的方式避免了直接对抗。

此外，DeepMind这家公司还特别神秘，觉得自己肩负改变世界的责任。所以即便是去这家公司面试，你也拿不到具体地址，只能到附近地铁站，再由专人从地铁站接到公司。

2014年，谷歌花了6.5亿美元收购了哈萨比斯的DeepMind。这个价格在当时相当惊人，因为DeepMind还没有什么商业化产品。哈萨比斯提出两个条件：一是不允许把DeepMind的成果用在军事用途，二是收购方必须成立伦理安全委员会，防止AI被导向失控的未来。谷歌全都答应了。

被收购后的DeepMind生活一下子好起来了，能用高薪吸引很多学者来做研究，而且对这些科研人员的照顾更是无微不至——比如员工食堂堪比迪拜五星级酒店的自助餐，科研人员待遇堪比摇滚明星。有研究员曾经给支持部门发邮件，说如果把草莓上的叶片摘掉会更省时间。两天后，自助餐桌上就摆上了处理得干干净净的草莓，不带一点绿色。

**但生活条件好了，哈萨比斯的理想却遭遇了阻碍。**书里在描写那几年时，一个主线让我印象很深：理想主义的哈萨比斯一直想要推动伦理委员会的成立，可谷歌根本不想让这个委员会真正运作，它最关心的只有挽救低迷的股价。

2015年，奥尔特曼和埃隆·马斯克等人成立了OpenAI。他们在官网列出了7位创始人，宣称要做“非营利”AI研究，让AI造福全人类。

哈萨比斯看到这个消息时，整个人震怒不已。他发现，OpenAI的7位创始人里，有5个人曾经在DeepMind担任过几个月的顾问或实习生。这5人在DeepMind的时候，哈萨比斯对他们毫无保留，把实现通用人工智能的策略全告诉了他们，包括怎么构建自主代理、怎么教AI玩国际象棋和围棋、怎么设计神经网络架构。

结果呢？这些人转头就成立了一家竞争对手公司。书里没详细写哈萨比斯当时的反应，但不难想象那种感受，就像你掏心掏肺教出的徒弟，结果他们转头在你的对面开了店，还用你教的手艺跟你抢生意。不过，哈萨比斯的愤怒，可能不只是因为“背叛”，更是因为他发现自己当初坚持的“伦理委员会”，在这场资本驱动的竞赛面前，显得那么无力。

奥尔特曼那边呢？他可能觉得自己做的是对的。**AI技术不该被一家公司垄断，应该有更多的竞争者，应该有不同的技术路线。**但问题是，当你用“非营利”名义成立公司，然后去挖竞争对手的人，这件事本身就很微妙。

这场"背叛"，成了两个男人之间矛盾的起点。而更大的冲突，还在后面。

2022年11月30日，OpenAI发布了ChatGPT。这次发布突然到什么程度？连OpenAI内部的很多员工都不知情，包括部分安全研究员。有些员工甚至打赌，一周后会有多少人使用它。他们预计最多10万用户，毕竟它看起来就是个带文本框的工具。结果ChatGPT在5天内就突破了100万用户，两个月后突破1亿用户。

谷歌最初没太在意，高层对那些“ChatGPT会替代谷歌搜索”的说法毫不在意。但这种淡定只持续了不到一个月。

2022年12月，谷歌CEO桑达尔·皮查伊在公司内部发布了**“红色警报”**（Code Red）。在硅谷科技公司，“红色警报”代表的是当前、紧急、直接的危机，通常用于真实生产系统的重大事故，比如整个搜索或Gmail产品突然宕机。一旦发布红色警报，修复代码可以立即推到生产系统，不用走测试审核试运行等常规流程。这种警报不会持续太久，问题解决就取消，最长也就几天。

但这次不一样。**这次的红色警报，并不是谷歌的生产系统或产品真的遇到了问题，而是一次带有比喻性质的红色警报。**皮查伊想告诉员工，必须要把ChatGPT给谷歌带来的威胁，当成谷歌搜索宕机一样严重的事来对待。

更戏剧性的是，谷歌联合创始人谢尔盖·布林和拉里·佩奇被紧急召回，亲自下场参与工作。这两位早已不插手日常事务的创始人，为了应对危机重新回归。谷歌员工内部开始吐槽。有人说，这是“起大早赶晚集”。

2023年2月7日，就在谷歌准备发布Bard的前一天，微软抢先宣布它的搜索引擎必应将迎来大规模AI升级。第二天，谷歌在巴黎召开发布会，正式展示Bard。按谷歌的宣传，Bard不仅能像ChatGPT一样有问必答，还更“负责任”——这在暗搓搓地指ChatGPT里掺杂太多虚假信息。

但是，Bard在唯一的演示环节翻车了。有人问：“关于詹姆斯·韦伯太空望远镜（JWST），我可以告诉我9岁的孩子它有哪些新发现？”Bard回答：“韦伯望远镜拍摄到了太阳系外行星的第一张照片。”

这个答案是错的。事实上，第一张系外行星照片是欧洲南方天文台在2004年拍摄的。

最先指出这个错误的是天体物理学家格兰特·坦布雷。他在推特上（现在的X）写道：“我不是要成为一个讨厌鬼，我相信Bard会令人印象深刻，但郑重声明：韦伯望远镜并没有拍下'我们太阳系外行星的第一张图片'。”

要知道，**这不是普通的AI回答错误，而是谷歌的类ChatGPT产品的首次公开亮相。**错误消息一出，谷歌股价大跌7.68%，一天内蒸发约1056亿美元市值。当时有网友评价：“这是AI到目前为止犯下的一个最昂贵的错误。”

谷歌不得不快速撤下了相关演示的YouTube视频。有员工对媒体说。谷歌这两年的各种活动往往追求市场宣传效果，放放产品视频就好，不管到时候展示结果如何。“产品和运营都麻痹了，太缺乏实战经验了，总有一天要出糗。只是没料到在跟微软硬抗的时候掉链子。真是天意。”

2023年4月，谷歌宣布将DeepMind和Google Brain合并，成立Google DeepMind，由哈萨比斯担任CEO。

可能在我们这些普通人看来，这不过是同一家公司里的两个团队合并，没什么大不了。但读过这本书，你就会知道，这两个团队虽然看起来都在研究AI，但是方向和风格却完全不同。

**这两个团队常年争夺顶尖研究员和计算资源，团队文化更是天差地别**。Google Brain更贴近谷歌主体业务，直接服务于谷歌产品改进，最早由吴恩达创立；DeepMind则侧重基础研究和学术突破，保持相对独立性。在合并前，两个团队长期明争暗斗。

但现在，谷歌顾不上这些了。ChatGPT的冲击实在太大，**谷歌只能放下内耗，整合所有资源来应对挑战**。

哈萨比斯开始领导新组建的Google DeepMind，并负责监管大语言模型Gemini的开发。2023年12月，谷歌终于发布Gemini 1.0，哈萨比斯亲自主持发布会，称它是谷歌“迄今最先进、最强大的AI模型”。

但市场当时的评价很直接，有媒体说：如果说Gemini 1.0是一次仓促的追赶，那它确实只做到了——仓促。

在接下来的一年多时间里，哈萨比斯和团队一直在憋大招。可**每次谷歌准备发布新产品，OpenAI总会“碰巧”抢先一步，这个“碰巧”得打引号**。

2024年2月15日，谷歌准备发布Gemini 1.5 Pro，这是一个支持100万tokens的多模态大模型，性能远超GPT-4 Turbo。结果同一天，OpenAI发布了文生视频大模型Sora，**全网关注度瞬间被Sora吸走了，Gemini 1.5 Pro在大模型发展历史上的意义被严重低估。**

2024年5月14日，谷歌要开I/O开发者大会，结果前一天OpenAI突然搞了场“春季更新”活动，发布了GPT-4o。更绝的是，奥尔特曼还在X上发帖，贴上了OpenAI活动现场以及谷歌发布会现场的照片，并配文“我尽量不去过多思考竞争对手，但我却无法不去思考OpenAI和谷歌之间的审美差异。”这话很直白，就是在抨击对方的审美和品味。

**这种“狙击”策略，OpenAI用得炉火纯青。**每次都卡在谷歌发布会前一两天，甚至干脆同一天，把谷歌的风头抢得一干二净。

2024年12月，谷歌终于反击了。12月5日，OpenAI开启“12天连续发布”活动，每天推出一个新功能或新产品，搞得整个科技圈的目光都盯在它身上。就在OpenAI活动进行到第五天时，谷歌突然发布了重磅更新Gemini 2.0 Flash，这也是Gemini 2.0系列的首款模型。

这是谷歌的“反狙击”，国内媒体当时用“深夜炸场”“截胡式发布”来形容。有评论说，谷歌这一招让OpenAI的连续营销“直接付之东流”。

但真正的逆袭，其实发生在2025年11月18日。谷歌在没有任何预警的情况下，发布了Gemini 3 Pro。

虽然现在还没法说接下来哪家AI会赢，但我想说的是，这次Gemini 3 Pro的发布就像拳台上一直被压着打、抱着头躲避的蓝方拳手，突然抓住一个机会，用一记有力的右勾拳将红方拳手击倒在地。

然后，就出现了开头的一幕：奥尔特曼给Gemini 3 Pro的发布点了个赞。

**这个点赞可能是认可，也可能是无奈。**从2015年OpenAI成立到2025年Gemini 3 Pro发布，十年时间，两个男人的战争从未停止。哈萨比斯从最初的愤怒，到被收购后的妥协，再到被ChatGPT冲击后的慌乱，最终用Gemini 3 Pro证明了自己。

当然，在《至高无上：一场颠覆世界的人工智能竞赛》这本书里还有更多我没讲到的细节，比如Transformer论文的8位作者为何全部离开谷歌，又比如奥尔特曼2023年11月被解雇又火速复职的内幕，再比如微软、马斯克和DeepMind联合创始人苏莱曼之间的复杂关系。

这本书的作者帕米·奥尔森是彭博观点专栏作家，从事科技行业报道超过13年。**她用大量第一手采访和内部资料，把这场AI竞赛写得像悬疑小说一样精彩**。如果你想理解AI竞赛背后的人性、野心、妥协和资本逻辑，这本书值得一读；就算你只是想了解一些AI发展的幕后故事和人物，这本书也不会让你失望。
