# 624｜三大AI公司扎堆交作业：DeepSeek像蜜雪冰城，Claude像喜茶

2025年10月1日

你好，我是快刀青衣。欢迎收听快刀广播站，每天带你看AI。

昨天，也就是9月30号，本来一心期待国庆中秋双节假期的我，被一连串的AI领域消息冲击着。

最核心的就是这三条：

- **深度求索发布了DeepSeek-V3.2-Exp**，与之前版本相比，API价格直接削减了75%。
- **OpenAI给ChatGPT加上了购物功能**，可以直接在聊天窗口买东西了。
- AI编程领域的王者Anthropic继续卷出新高度，推出了**Claude Sonnet 4.5新模型，号称能连续工作30小时写代码。**

好家伙，我在想，这三家公司的创始人是不是有个小群，一个人在群里说，咱们放假前搞点事情吧，不要让这帮打工仔好好过节，剩下两个人积极响应。要不怎么都赶在国庆中秋假期前一天，第三季度的最后一天集体“交作业”。这有点儿像内娱明星们发自己的负面公开信，通常会集中在周五下班后，偷偷发条小微博，用来降低媒体的关注。

不过这三个产品发布的最新动态，每一个都非常有特色，这让我想起1998年新华字典里那个经典句子：“张华考上了北京大学；李萍进了中等技术学校；我在百货公司当售货员：我们都有光明的前途。”

说实话，这句话当年是用来解释冒号用法的，出现在1998年《新华字典》修订本第673页。之后的一版中，才把“光明的前途”改成了“光明的未来”。

我来套用下这个句式，那就是：**DeepSeek搞出了稀疏注意力机制；ChatGPT上线了购物功能；Claude能连续编程30小时：它们都有光明的未来。**

从这一天的发布内容上，已经能看出这三家公司的个性化特点。我突然觉得这三家公司，非常像三家知名的连锁品牌：

- **DeepSeek像蜜雪冰城**，主打性价比之王，价格便宜量又足，加上开源机制，搞得大街小巷到处都是。
- **ChatGPT有点像星巴克**，全球化最完善，商业化包装精美，从咖啡到三明治，再到杯子和月饼，总有拿下用户的产品。
- **Claude有点像喜茶**，从编程到写作到使用界面，整个品牌感觉很有审美和档次。

那咱们一个一个来看看这三家公司，在假期前到底交了什么“作业”。

先说说这次发布的**DeepSeek-****V****3.2-Exp**。先解释一下这个命名，Exp三个字母代表的是“实验版本”，表示这是一个技术验证性质的版本，主要是用来探索新一代架构的可行性和优化方向。

这个版本最大的亮点就是那个**“稀疏注意力机制”**，简称DSA，听起来很学术也很高大上，其实原理挺简单的。

你想想，传统的AI就像一个人同时关注房间里的每一个物品，哪怕是墙角的垃圾桶也要仔细看。用这种方式，特别是房间里东西很多的情况下，就需要耗费大量AI算力来跟踪每一个物品。而DeepSeek的稀疏注意力机制就像一个聪明人，先快速扫描一遍，找出最重要的几个东西，然后只专注于这些重要物品进行深度思考。

从技术层面来说，DeepSeek把这个过程分成了两步：

**第一步是“闪电索引器****”**，先快速扫描上下文，就像店员快速扫一眼你的表情，大概知道你想要什么。

**第二步是“细粒度的标记选择机制”**，只对筛选出的关键信息执行全维度计算，就像专门给你推荐合适的几个商品。

这样一来，计算复杂度就可以指数级下降。用大白话说，就是原来需要100个步骤的事情，现在只需要10个步骤就能搞定，而且效果基本不变。

**这个技术的目标是****保****证性能和成本的相对平衡**。对咱们普通用户来说，最直接的好处就是价格下降。DeepSeek这次API价格降幅最高达到75%，输出价格从每百万tokens12元降到3元，输入价格从4元降到2元，输入价格（缓存命中）更是从0.5元降到0.2元。

![](https://piccdn2.umiwi.com/uploader/image/ddarticle/2025093019/1888959479725246912/093019.webp)
    

这个价格有多震撼？咱们来对比一下最新的市场定价：

字节豆包的通用模型Doubao-1.5-pro-32k，每百万tokens输出价格是2元，输入价格是0.8元，输入价格（缓存命中）为0.16元。DeepSeek每百万tokens输出价格和输入价格，只比豆包稍贵一点。当然，如果是企业用户，豆包的这个价格仍可能享受折扣，也就是说，实际算下来，豆包的API可能更便宜，毕竟豆包背后的字节实在太有钱了。因此，未来AI的API接口价格战，很可能就在豆包和DeepSeek两家公司之间进行。

再看国外的竞争对手：我们先拿ChatGPT举例。最近刚刚降价的GPT-5，每百万tokens的输入价格是1.25美元，输出是10美元。上面这些数字大家无需特意记住，简单来说，按当前汇率计算，每百万tokens输入价格上，ChatGPT大约是DeepSeek的4.5倍，输出上的差别就更大了，ChatGPT是DeepSeek的20多倍。

所以对咱们普通用户或者中小型公司来说，这意味着用AI的成本真的大幅下降了。特别是需要大量文本处理的场景， 按照DeepSeek现在的研究方向，未来成本将继续有大幅度的降低。

再看OpenAI，这次ChatGPT的更新就很有意思了——**不只是聊天工具，还要做购物平台**。

**这个“即时结账”功能听起来挺科幻，其实就是把购物流程整合到聊天里**。你跟ChatGPT说“我想要100美元以内的陶瓷乔迁礼物”，它立马给你推荐商品，配上图片、价格，还有购买链接。看中了直接下单，支付、配送、售后都在ChatGPT里搞定。

**但****关键不是这个功能本身，而是背后的商业逻辑**。OpenAI这次直接打通了Shopify，这可不是一般的合作。

Shopify是什么？它是仅次于亚马逊的北美第二大电商平台。更重要的是，Shopify代表的是“独立站”模式，也就是商家不用依赖淘宝、京东这样的大平台，可以搭建自己的电商网站。

现在OpenAI和Shopify合作，意味着什么？意味着全球几百万商户的商品，都可能通过ChatGPT直接触达用户。**这就不是简单的购物功能了，****这是****在重构电商的流量入口**。这个功能往后发展的话，估计会让现在的电商老大亚马逊非常难受。

**并且这种合作，可能意味着AI时代的很多电商逻辑都会发生根本的变化。**传统电商是这样的：用户想买东西，打开淘宝或京东，搜索商品，比价，然后下单。现在可能变成：用户跟AI聊天，让AI理解需求，然后AI直接推荐商品，一键下单。

这个变化有多大？想想看，ChatGPT现在每周活跃用户超过7亿，如果其中哪怕1%的用户通过ChatGPT购物，那也是700万用户的购买力。**而且这些用户的购买意图更精准，因为是通过自然语言表达需求的。**

**更厉害的是商业模式**。每笔通过ChatGPT成交的订单，商家都要给OpenAI分成。这就像在传统电商之外，又开辟了一个新的流量渠道，而且这个渠道的转化率可能更高。

加上前几天他们刚上线的AI智能报告Pulse功能，可以发现**OpenAI最近一直在C端用户的需求场景上下功夫**。对普通用户来说，这些功能让ChatGPT从一个聊天工具变成了一个生活助手。需要什么直接说，AI帮你找、帮你买、帮你安排。

最后说说Claude的这次大版本更新，这次Sonnet 4.5的更新就很有“工匠精神”的意思——**追求品质，注重工艺，有点工作狂的感觉**。

**最震撼的是那个“连续30小时编程”的能力**。你想想，一个程序员连续工作30小时会是什么状态？累得眼冒金星，代码质量肯定下降。但Claude不一样，它能保持高质量地连续工作30小时，写出11000多行代码，还能构建出完整的聊天应用。

技术上，Claude Sonnet 4.5在编程基准测试SWE-bench上达到了82%的准确率，超过了GPT-5的72.8%和Gemini 2.5 Pro的67.2%。这个测试是衡量AI解决真实世界编程问题的能力，82%意味着10个编程任务里能正确完成8个，这已经是相当高的水平了。

更厉害的是在另一个OSWorld测试中，从前一代模型的42.2%跃升到61.4%。OSWorld是测试AI操作计算机系统的能力，**这个提升意味着Claude不只能写代码，还能像人一样操作电脑完成复杂任务。**

这次更新还有个Agent SDK，让开发者可以基于Claude构建各种智能体。这个SDK解决了三个关键问题：智能体如何在长时间运行中管理内存，如何平衡自主性与用户控制，以及如何协调多个子智能体协同工作。

这几个更新，对全球任何一个开发团队来说，都是一个重大的利好消息。因为对程序员来说，这**意味着有了一个真正靠谱的编程助手，不只是简单的代码补全，而是能理解复杂需求、处理多文件项目、进行长时间开发的高级助手**。

最后，从这三家公司的更新可以看出，AI行业头部产品正在分化出不同的发展路线：

**DeepSeek走的是“普惠路线”**。通过技术创新大幅降低成本，让更多人用得起高质量AI服务。就像蜜雪冰城让奶茶从奢侈品变成日常消费品一样，并且把奶茶店开满了大街小巷。

**ChatGPT走的是“生态路线”。**不只提供AI能力，还要构建完整的服务生态，从聊天到购物到家庭管理，一站式解决用户需求。就像星巴克从咖啡店变成生活方式品牌一样。

**Claude走的是“专业路线”。**专注于提升AI的专业能力和工作质量，服务那些对品质有高要求的用户。

说实话，看到这三家公司在假期前一天集体发布更新，我想到的不只是技术竞争，更是对用户需求的不同理解。有人觉得AI应该便宜好用，有人觉得应该功能全面，有人觉得应该专业精准。

李飞飞教授曾经在接受媒体采访时，说了句被引用很多次的话，她说：“此刻我认为我们正经历着**一场堪比寒武纪大爆发的增长**，这几乎可以说是爆炸式的。 不仅在文本处理方面，你还能看到图片、视频、音频等各种AI应用和模型的涌现。这是一个极其振奋人心的时刻。”

要知道在生物学历史上，“寒武纪大爆发”是地球生命演化史上最关键的事件之一。在地球的时间尺度上，寒武纪短短2,000万到2,500万年之间，几乎所有现代动物门类的祖先都突然地涌现了出来，一下子呈现出了生物多样性。

我们现在也是一样，从底层模型到AI应用，每天都有新的产品出现，也许这就是AI时代的魅力：**不是一家独大，而是百花齐放，每个人都能找到适合自己的那一款。**

趁这个假期，你不妨找一个一直想学习的AI工具，开始每天试着用一用，看看是不是符合你的需求。毕竟，AI工具再好，最终还是要服务于我们的生活和工作。
