# sources

把已確認值得長期保留的原始課文放這裡。

建議：

- 按年份分目錄，例如 `raw/sources/2025/`、`raw/sources/2026/`
- 單篇檔名優先使用 `broadcast_id.md`
- 若來源沒有穩定編號，再使用 `special-YYYYMMDD-<slug>.md`

例子：

- `raw/sources/2026/812.md`
- `raw/sources/2026/786.md`
- `raw/sources/2025/353.md`

原始 workspace path 請留在檔案 frontmatter 或對應 broadcast card metadata 中。
