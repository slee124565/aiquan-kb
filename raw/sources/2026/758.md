# 758｜教程上新：让专业经验更好复用，两节课带你上手Claude Code

12分28秒

转述师：AI

你好，我是快刀青衣。欢迎收听快刀广播站，每天带你看AI。

最近这两个月，我相信很多同学打开各种社媒软件，会发现里面有大量的最新AI工具，例如Claude Code能自动写代码，小龙虾OpenClaw能操控电脑，每个看起来都神乎其神。评论区里一片“太牛了”、“要失业了”、“必须学”。

说实话，我特别理解评论区里的这种焦虑。AI最近半年的技术突破确实很快，新工具 、新产品层出不穷。但我观察下来，发现了一个更关键的问题：这些技术远没到能席卷一切的程度，因为我们每个人的现实生活都足够复杂。

更让我在意的是，很多人都容易陷入一个怪圈，那就是100种工具都“用过”。他们说起每个工具的能力来，都头头是道。但如果问他用这个工具做出了什么，却发现自己一个作品都没做出来。

那这个“用过”是什么意思呢？就是看了教程、下载了软件、跑通了“Hello World”，然后就没然后了。再看到新工具，又重复一遍。看起来紧跟前沿，实际上什么都没沉淀下来。

今天我想和你推荐的，是本周王树义老师在咱AI学习圈新上线的一门教程课，帮你用Claude Code快速做科研。

相信咱们圈子的老同学对王树义老师很熟悉。王老师的本职工作是大学老师，不过在我看来，他对新技术的应用和分享，肯定超过了90%的大学老师，因为几乎每个新工具出来，他都会自己先用，然后录成教程分享给大家。我相信他在互联网上的学生数量，肯定远远超过了他在大学校园里的学生。新上线的这门课，核心就是带你快速上手Claude Code。如果你了解最近AI领域的发展，那么对Claude Code一定不陌生，很多AI写代码，包括爆火的小龙虾的内核，其实都是Claude Code。

而关于Claude Code的教程，其实早在2025年10月的时候，我就想给咱们同学推出了，但是当时确实门槛非常高，无论是技术门槛还是费用门槛。不过随着技术的迭代，今天王树义老师将带你用一种非常简单的安装方式搞定Claude Code，而且对于不少同学关注的费用问题，王老师也会采用性价比非常高的国内模型做平替。

但我和你推荐这个教程，不是为了让你“又多学一个工具”。恰恰相反，我是**希望你花一个小时的专注时间，把一个工具真正用起来，并且做出点东西。**

要注意的是，哪怕王老师把很多环节都拆得很细了，Claude Code还是有使用门槛的。因为每个人的电脑和环境都不一样，安装过程会遇到各种问题，会看到程序员用的那种黑底白字的终端界面，可能会让你觉得“这是不是太难了”。但正因为有这个门槛，当你调通的那一刻，成就感会非常强。

更重要的是，在这个基础上，你就可以用AI做很多真正的工作了——**不是“研究过”，而是“用起来了”。**

其实我们平时很多用AI的方式，挺低效的。每次有任务，就打开ChatGPT或者豆包，写一段提示词，可能300字，描述清楚要做什么、输出什么格式、注意什么细节。AI给你结果，这次任务结束。

然后，下次遇到类似任务，得再来一遍。又是几百字提示词，又要重新描述一遍需求。

而今天，王老师教你的方法，就是让你在Claude Code的环境中，能把你已有的经验复用起来。怎么做到呢？这就要引入一个新名词：**Claude Skill**，你可以把它理解成“给AI看的操作手册”。

第一次操作时，你需要花时间把整个工作流程“教”给AI，包括目标是什么、分几步做、每步注意什么、输出什么格式。这个过程确实有门槛，可能需要你反复调试。但一旦调通，下次你只需要一句话：“用那个文献综述的技能，帮我整理一下最近的行业报告。”AI就知道怎么做了。

我自己试过，拿王老师教的方法，安装好一个文献综述的Skill，花了大概10分钟。但之后每次用，都只需要输入一句话，AI就知道怎么做。这10分钟的投入，可能使用两三次就回本了。

这就是可复用的价值——**你的专业知识和工作流程，真的被沉淀下来了，而不是每次都从零开始**。更关键的是，这个Skill可以分享给同事、学生，甚至可以放到社区里给更多人用。

说到这里，可能有同学会问：那跟着王老师这个教程学完，到底能学到什么？虽然标题叫“如何做好学术研究”，但我相信咱们很多同学跟科研没什么关系。不过里面的案例，**完全可以迁移到你的工作和生活场景。**

王老师在教程里重点讲了两个案例，我分别说说。

第一个是《红楼梦》人物关系图谱。AI自动做网络调研，收集信息，然后生成一张专业的可视化关系图，荣宁二府、金陵十二钗、所有人物的关系都标得清清楚楚，最后输出成PDF。

这个能力可以迁移到哪些场景？

首先，**在职场里，几乎所有需要出深度报告的人都用得上**。比如你要写一份行业分析报告，需要梳理产业链上下游的关系、竞争格局、资本关系，用这个工具就能快速生成一张关系图谱，比你手工整理快太多了。

再比如你刚进一家大公司，组织架构复杂，各个部门之间的协作关系搞不清楚，用这个思路把公司架构和业务关系可视化，上手速度能快很多。

**如果你是家长或者老师，这个工具同样****有****用****。**你可以给孩子做学科知识的思维导图，比如历史课讲到战国七雄，让AI生成一张各国之间的外交关系、战争关系图谱，比你口头讲清楚得多。老师备课的时候，也可以用这个工具快速生成知识图谱，这不就是历史课上拿来即用的素材吗？

我跟着王老师做了一个林冲逼上梁山的人物关系图，我放在文稿末尾的附录里了，你可以看看。这样的图表，在以前我们如果要自己画，至少需要一个小时，并且还要对内容非常熟悉，但是现在，两分钟就全出来了。

说完关系图谱，我们再看第二个案例，这个更震撼。

王老师会带你用一个叫“心理学文献综述”的Skill，从259篇论文里筛选高引用文献，自动生成符合APA第7版格式的综述文档，包含摘要、引言、PICOS框架等完整学术结构。整个过程，AI自动处理数据库访问限制，遇到问题还会调整策略。

王老师给了一个时间对比，我看到的时候整个人都愣了一下：过去做这件事需要3天到一周的时间，现在不到15分钟。你看，15分钟和3天比起来，**这不是效率提升，这是降维打击**。而且Skill一旦做好，你身边的所有人都能用。

当然，**这不仅仅是绝对时间的节省，更是极大地扩展了我们每个人的能力圈。**

像之前，我们写任何文章或者分享的时候，很少会花一周的时间去研究一下最新的论文进展，但是现在，当你让你的AI学会了这个技能，那么相当于你在研究任何话题的时候，都可以去看看最新的学术研究。比如我就跟着王老师的教程，做了一个社交型AI与儿童社交行为的心理学论文综述，在里面发现了不少我之前根本不知道的研究，我也把这个结果放在文稿末尾的附录里了。

再扩大一步，**这个能力迁移到职场，就是所有需要“查资料+整理+成文”的场景。**

比如你要写一份竞品分析报告，需要查50篇行业报道、10份研究报告、20个产品评测，然后提炼核心观点、整理成文档、统一格式。这个过程用传统方式，少说两三天。用这个Skill，可能一个下午就搞定。

再比如你是律师或者咨询顾问，经常需要整理大量案例或者项目经验，写成专业报告。这种“重复性高但又需要专业判断”的工作，正是Claude Skill的强项。

当然，说到这里，就不得不提Claude Code的门槛，我知道很多人会说：“我是文科生，这个我没学过。”但你想想，大学时的所有课程，不都是你上高中没学过的吗？为什么那时候你可以从零去学一个新东西，现在反而要拿这种高中的文理分科当幌子？

像我就是绝对的理科生，在我们公司年会上做答题游戏的时候，当被问到“颜筋柳骨”指的是谁时，“颜”我答了出来了，但是后面的“柳”，我的脑海里一直盘旋着另外一个名字“柳宗元”，正确答案是“柳公权”。旁边同事哄笑，都说我“居然连常识都不知道”。我为什么举这个例子呢？你想想，如果在当时，我跟大家说“我是理科生，这个不记得也正常”，那是不是就是自己在骗自己呢？

说到底，**这不是能不能学会的问题，是愿不愿意开始的问题。**但正因为有这个门槛，当你调通的那一刻，成就感才会这么强。

而且这个门槛其实也没想象中那么高。王老师在教程里把整个安装流程讲得非常细，比如macOS怎么装、Windows怎么装、遇到常见问题怎么解决，每一步都有截图、有命令、有解释。

更关键的是，你不是一个人在战斗，遇到问题你有AI帮忙。我的建议是：你可以一边安装，一边开个豆包或者其他AI工具，遇到任何报错都把错误信息截图或者复制进去，问“这是什么问题，怎么解决”。豆包会告诉你下一步怎么办。AI时代解决技术问题的方式，就是用AI。

最后，我想说的是：**别再观望了。**

我知道很多人的心态是“等准备好了再说”。但说实话，AI工具的迭代速度，已经不允许你“等准备好了再下水”。你要是等着“准备好”，那就永远在岸上看别人游泳。

所以，我的建议是：花一个小时，就现在，把Claude Code装上，跟着王树义老师的教程走一遍流程，做出一个作品来。

这里，我还要必须强调一点：**AI学习圈的教程，最重要的不是“看”，而是“做”。**

很多人的习惯是看完教程，留完言，然后就没然后了。但你不跟着一步步操作，永远体会不到那种“调通了”的感觉。

哪怕最后做出来的东西不尽如人意，但当你真的做出来的时候，你会发现：这东西没那么恐惧和遥远了。

这一个小时，你可能会遇到报错，可能会卡壳，可能会怀疑自己。但没关系，把问题抛给AI解决。当你真的生成了第一张关系图谱、或者完成了第一份文献整理，那种成就感会告诉你：这一个小时，比你刷100条短视频、研究100种工具有用得多。

而且这只是开始。一旦你掌握了Claude Skill的用法，你会发现很多工作场景都可以用这个思路来做，把重复性的、有规律的工作流程“教”给AI，让AI记住，下次一句话调用。

**你的工作效率会发生质的变化。**

只有下到游泳池里，你才知道自己会不会游泳，甚至可能发现自己有游泳的天赋。站在岸上研究理论，永远学不会。AI时代也是一样。
